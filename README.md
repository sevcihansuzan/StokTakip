# Endüstri 4.0 Üretim Takip Sistemi
Bu proje, MQTT protokolü üzerinden gelen sensör verilerini anlık olarak izleyen ve 
veritabanına kaydeden bir WPF (.NET) uygulamasıdır.

## Özellikler
* **Anlık İzleme:** OxyPlot ile canlı verim grafiği.
* **Haberleşme:** MQTTnet kütüphanesi ile endüstriyel veri takibi.
* **Yetkilendirme:** Kullanıcı bazlı sayfa erişim kontrolü.
* **Veritabanı:** IoT verilerinin geçmişe dönük saklanması.
